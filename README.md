## Getting started

- Visit [the live site](https://chuusa.vercel.app/)
#### OR
- Clone this repository
- Create a .env.local file in the root
- Add the variable  ``` API_END_POINT_LOCAL=https://chuusa-api.herokuapp.com/api/v1/convert ```
- Run ``` npm run dev ```
